# 📘 School Management System

A **School Management System** developed using **Java** and **Maven**, built in IntelliJ IDEA.  
This project aims to streamline and automate essential school operations such as managing students, teachers, courses, and attendance.

---

## 🚀 Features

- 👨‍🎓 Student Management (Add, Update, View Records)  
- 👩‍🏫 Teacher Management  
- 📚 Course / Subject Management  
- 🗓️ Timetable Organization  
- 📝 Attendance Tracking  
- 📊 Basic Reporting System  

---

## 🛠️ Tech Stack

- **Java**
- **Apache Maven**
- **IntelliJ IDEA**
- Object-Oriented Programming (OOP)

---

## ⚙️ Getting Started

### ✅ Prerequisites

- Java JDK 8 or higher  
- Apache Maven  
- IntelliJ IDEA (recommended)

---

### 📥 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/caintcreation/school-management-system.git
````

2. Navigate into the project folder:

   ```bash
   cd school-management-system
   ```

3. Open the project in IntelliJ IDEA

4. Build the project using Maven:

   ```bash
   mvn clean install
   ```

5. Run the application:

   * Locate the main class
   * Right-click → **Run**
